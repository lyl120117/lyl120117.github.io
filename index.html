宿命星盘

玩家上线时赠送星盘秘匣1个，可以开出对应职业的星盘，另赠送9999个星魂百纳丹，可将星盘免费升至满级。

星盘---本命星、天命星

  1. 每个星盘均有10个星位；
  2. 位于内侧的5个是本命星，位于外围的5个是天命星；
  3. 星位亮起时，表明该星位处于激活状态；
  4. 点击激活状态的星位，可以查看其对应的激活属性；
  5. 通常情况下，在天命星位置激活的属性强度会比较高。

资质

  1. 每个星位上的数字表示该星位的资质；
  2. 资质会影响每个星位激活的属性强度，资质越高属性效果越强；
  3. 总资质上限位20.00；当总资质未达到上限时，每个星位的资质可能不会全部生效；
  4. 单个本命星的资质上限为5.00；
  5. 天命星的资质等于相邻两个本命星的资质之和。

等级

  1. 星盘左侧的蓝色进度条表示星盘的等级和当前经验；
  2. 星盘等级会影响星盘所有属性的强度，等级越高属性效果越强。

星噬

  1. 星噬操作可以吞噬其他的「星盘」或「星魂百纳丹」为当前星盘增加经验；
  2. 吞噬其他星盘时会获得该星盘积累的全部经验；
  3. 吞噬其他星盘时可以选择保留星盘资质或继承新星盘资质。

占星

  1. 占星操作会改变星盘激活的星位数量、属性内容、属性位置；
  2. 星盘的资质、经验、等级会得到保留。
  3. 占星操作需要消耗「星云散聚石」，部分「星云散聚石」会提供星运经验。

星运

  1. 星盘右侧的紫色进度条表示角色的星运和当前的星运经验；
  2. 星运和星运经验与角色相关，不会因为更换星盘而发生改变；
  3. 星运决定了角色「获得星盘」或「占星」时激活的星位数量；
  4. 星运越高，激活更多星位的概率越高，激活的星位数量至少等于角色星运。

观星

  1. 观星操作会改变星盘激活的星位；
  2. 星盘激活的星位数量，属性内容均不会改变；
  3. 只有当相邻的两个本命星均处于激活状态时，天命星才可能被激活；
  4. 观星操作需要消耗「星行诡道散」。

星变

  1. 星变操作会随机为5个本命星中的1个增加资质；
  2. 总资质达到上限20.00后，再进行星变操作将会把所有资质重新随机分配到5个本命星上；
  3. 星变操作需要消耗「性灵奇引珠」。

诛天浮屠塔

诛天浮屠塔基本规则：

  1. 副本为单人副本，以单人闯关为目标，共设有108层关卡；
  2. 副本每周通关（108关）一次，每周三早上七点，副本将重置；
  3. 副本单日不限次，您可以随时随机进入副本开启闯关模式；
  4. 在一个周期内，玩家可随时进入，每次进入可以接着上一关的继续挑战；
  5. 副本为动态副本，您每一次进入都将开启全新的挑战模式。

诛天浮屠塔关卡规则：

  1. 您通关某一小关卡成功，领取奖励后，可点击屏幕左上方的按钮呼出选关界面；
  2. 开启选关界面后，点亮的标志为您已通过的关卡，灰暗的标志为未通过的关卡。您可对点亮的任意关卡进行选择，系统默认的关卡为您最新开启的关卡；
  3. 您每进入一层关卡，激活关卡后，系统开始计时。界面将显示您通过的关卡时间。通关成功后，您可以通过奖励界面查看您的通关时间。

诛天浮屠塔奖励规则：

  1. 道具奖励：您每通过一层小关卡，将会获得道具和积分奖励。道具奖励为周一次性奖励，玩家获取奖励后，本周内再次通过该关卡将不再获得奖励，但仍可获得积分奖励；
  2. 道具奖励在闯关成功时会通过界面进行发放。根据每一阶段的难易，参与抽奖的道具种类及数量均有不同。挑战层数越高，奖励越丰厚；
  3. 您下线、离开副本等行为不会影响奖励。您可以在任何场景通过奖励界面领取您未领取的奖励；
  4. 一旦您再次进行选关，未领的奖励将会被覆盖。并且本周内将无法再领取到该关卡的奖励；
  5. 积分奖励：您闯关成功可以获得积分，积分可以通过界面直接换取想要的BUFF，加快过关进度。此类BUFF离开副本后消失。
  6. 可获得的道具奖励包括：星云散聚石，星行诡道散，星灵奇引珠，五铢钱，高级五铢钱，流银币，锦囊·无双/无畏/无敌（限时15天）。

诛天浮屠塔面向等级不低于89级，修真达到上仙/狂魔的玩家开放。
此外，诛天浮屠塔副本还设有勇士排行榜，按勇士单关闯关成功的累计最短时间进行排行。该时间为玩家本周内第一次闯过每一小关卡的累计时间，排行榜为每一个小时刷新一次，每周三早6点排行榜封榜。
 勇士排行榜奖励设置：（以封榜时间排名为准）第一名奖励5千元宝，第二名奖励4千元宝，第三名奖励3千元宝，第四名奖励2千元宝，第五名奖励1千元宝。（排名不分职业，以总排名为准，奖励领取时间每周三，过时视为自动放弃）。

幻沙蜃境副本攻略

开启副本条件：2-10人
 副本位置：埋骨之地，可从幻境之石传送进入
配置：SS、MM、WX、JO、法系、MG即可，MM则带复活卷以防万一，如死亡后没复活卷且MM都起不来，可小退再上线然后点回城再 点（是，同意进入副本）就会进入刚才挂的地方 如果在FB里回城是在FB门口。怪物说明：副本中的怪物组成包括（普通怪物、精英怪物、普通boss、终极boss）进入副本后，将会自动获得“世精血璃镜”和“忠义·关圣附体”锦囊各一个，拥有“世精血璃镜”可以穿越层层迷沙，按M键打开地图，看到原本隐于迷沙中的迷宫地图全貌。队长领取“智闯幻沙蜃境”任务（即魔魇神兵），队员可以领取每日任务“沙舞连环”（做了就无需在领了）。任务领取后，进去的起点，会随机一方的路北打开，即可开启征程。
 【小兵击杀流程】
 最新副本怪物组成为：3个小BOSS+1个大BOSS，每个BOSS都是随机，大BOSS一共3个，也是随机。
 小怪分为精英怪与普通小怪，名字带花边的是精英怪，BOSS的名字是金色花边，普通小怪是没有花边的。
 小怪属性：每个小怪都有不同属性，可分为2个大属性就是霸体和不是霸体。霸体的怪是不吃晕的，所以霸体的怪有5.0就5.0打吧。不是霸体的怪，如果队伍攻击不够，WX尽量控住，把他当成敌人晕死他吧。如果攻击够那么就集火打怪放龙、剑灵上剑印其
 他人集中打一个。
 【BOSS击杀流程】
 懈翼邪龙：
 开局所有玩家聚集在一起集火暴元打，MM开阵和解毒，同时站最远的角色会灵魂受到折磨，此时过1秒-2秒中毒，中毒会持续掉血，解毒的MM一定注意（毒是蓝紫色一个小脚的状态）。
 BOSS血量为50%的时候会出来2只老虎，一只黑虎、一只白虎，此时BOSS身上会免疫伤害，所以打老虎，顺序随意，击杀完两只老虎后，其他人暴元，WX等BOSS身上免疫状态消失后放龙。
 黑白虎死后大概2秒BOSS免疫状态会消失，等几秒后BOSS会将大家击飞，此时大家分散，小地图绿点不要重叠，抗BOSS的玩家不要撤退，其他人分散不要重叠也不要在BOSS正面太近，不然等BOSS喊分散躲避锋芒的时候，BOSS会回血，这个时候MM分散，照顾身边的队员，最远的还是会解毒，如果MM离的远不能及时解毒，要么点信念要么点6秒无敌吧，就这样把BOSS打死。
 元素末裔：
 这个上去第一下集火暴元WX龙MM开阵，MG把蓝色的冰碑打死，法系不要群，物理的可以群，打几下之后BOSS身上还会有免疫法系和物理系的状态，这个时候就攒元，物理系继续把出的蓝色冰碑打碎，等BOSS说破冰的时候其他人暴元WX龙开打，重复这样的，打死BOSS，如果队里法系多，可以冰碑火碑轮着打。
 三影罗刹：
 分配一个人，最好是MM去，去三影面前晃下然后其他人把仇恨拉过去都进木圈（绿色）如果仇恨已经拉过去了，那么分配出来的人，拉雷池就是把雷池拉到队员以及地上绿红蓝三圈之外不会群到队员的地方或者站在石头上就不会出雷池圈拉，然后BOSS每次会说进什么圈，木=绿色圈、水=蓝色圈、火=红色圈，没拉圈的MM在3个圈之间
 找个中心位置开阵阵能罩住大家，或者在木火圈之间开阵，这里进木圈和火圈都会中毒 水圈会被晕所以进水圈，开免疫和无极
 散吧，法系职业如果出的不是对应攻击属性的圈，就平砍BOSS，比如出木圈，FS就平砍吧。
 三障魔宗：
 黑屏会出好多一样的怪，所以还是一起上集火暴元、WX龙，如果有WS刚打的时候对BOSS放个生命禁锢，等黑屏出一样的时候，有生命禁锢的是真的，MM注意群加，BOSS击晕断阵，注意给开阵职业加血，如果没有WS，那么出怪的一瞬间用TAB切怪，切到怪的时候头上一瞬间有好多刚才放的状态的怪就是真的。
 【终极BOSS击杀流程】
 异界修罗：
 MM最好找个高的地方开阵，例如周外的骷髅架子上，去开阵其他人在下面打 SS拉过来让BOSS背对着开阵职业，另一个MM注意给受到破甲的人解毒，BOSS会一只出怪，分为物免和法免，所以要求其他人一定把物免和法免打死，如果打不死就会一个爆炸出2个，2个爆炸出4个，会把人爆炸炸死。
 迷魇之主：
 这个BOSS，SS拉过来一开始怪是法免的，物理系打BOSS，法系的打小怪BOSS是5.0的所以CK一只上荒追命，如果没学就上致命扫击，如果没CK有MO，MO就扔减速的小草，等法系的怪打完BOSS就会变成物免了。
 迦那婆罗：
 一个大蛤蟆上面坐2个美女的怪，这个和打流银宫的蛤蟆一样，扇形攻击，SS先上去然后转身咬BOSS，这个时候会有个火坑，让BOSS的脚和火炕平行，然后MG可以开神行，去BOSS面前离BOSS最远打，别太拉仇恨。然后1-2个人推灵魂，2MM分开站，一个负责专门给SS解毒加血，毒是蛤蟆的毒，中毒就得解，其他人可以在后面轻点打，如果是高攻没带F可以攒元，等WX有龙了上去暴元，走在BOSS面前吃12秒打BOSS。然后同时WX龙，这个时候高攻吃完12秒可以点绝对，WS可以开镜像，就是用一切保命的技能去打，打完死了，MM9再回去攒元，注意BOSS的毒死了还会有，要等毒没了再起来，下面说推灵魂，SS上去的时候，会在周围出2个鞭子，推灵魂的人负责挖，这个灵魂最好接力。
 第一下WX MG WS MO FS都可以，WX是用枪的流星赶月或者站火炕里用赤龙索，MG击退矢、MO花溅雨、WS土崩咒FS烈火炙翼（火鸟），如果灵魂离火炕近就需要鞭子去推，需要配合如果第一个人推的时候挂了，马上有人去上去帮推，推的时候如果去了BOSS前面就要点绝对了，要在固定时间内把灵魂推进火炕 失败一次就会让BOSS回满血驱除周围人身上状态。
 需要说的是地图的死胡同里有个会跳的箱子 是地图 可以在门口换镜子 有的时候挖地图会出现一个大漠盗贼 打死之后会掉材料，看人品吧！

诛天浮屠塔通关攻略

BUFF建议：除关数为9的倍数外，其他关卡开始触发前，最好全部加攻击和攻等，一般小任务完成后，再根据实际可重置积分。
 积分建议：对于装备比较一般的，后面多回到前面去刷积分。

关卡详述：

  1. 拯救信徒关卡！就是把僧人全部杀死就行，不能杀被蛊惑的信徒，单体技能打死就行，不要用群的，恶僧、妖僧、魔僧的buff技能很恶心，特别是后期的，建议准备好元，遇到中毒最好爆元。
  2. 传送门关卡！ 地面随机出光圈，碰触会出怪物1次10只，杀完100只任务完成。对于JO MG这类有阵的职业很好打，全部加攻击和攻等，先把10波怪全部引出去，然后上去开无敌状态，然后开阵。
  3. 纵横千胜！ 塔防游戏，中间有个佛珠，打佛珠加能量，NPC有2种，一种是摩休罗，群体攻击附带减速，伤害较低，另外一种安托罗，单体攻击，伤害高！进去就先K佛珠，然后第二波的时候，可以先把第三个NPC开为群伤的并升级第一和第三个NPC，然后继续打佛珠，buff毫无疑问全部加攻击和攻等。到后面这个挑战难度比较大，特别是107关，建议先升级第一个和第三个NPC为群攻，然后一直打到九波怪，就去继续升级其他的，boss出来了直接搞boss，攻击不够的，佛珠都没打完，不用管，只管揍boss。
  4. 五龙遁邪！ 五行属性的龙柱，随便打都可以，不过按照系统提示的去打，可以用2倍伤害，或者去掉负面状态！打完5个柱子任务完成，死亡就失败！如果抗的起就打2倍伤害，抗不起就先驱状态。
  5. 妖世灭佛关卡！4个方向出来4个四灵神兽，青龙（觉醒攻击伤害）白虎（觉醒物理防御）朱雀（觉醒法术防御）玄武（觉醒大量生命），首选打青龙，其次玄武，先全部加攻击和攻等，打boss的时候再重置重新加buff就是了，一般情况消灭青龙和玄武还是可以的。然后就是抗着打boss，看到boss说“妖佛之击”，最好爆元无敌，当然血量够就无所谓。但必须加血量上限，否则该技能容易秒人。
  6. 杀戮神荒关卡！BOSS无敌，触碰绿色光球可以暂时消除BOSS无敌状态，这个时候狂揍boss。孔雀明王的血快没的时候，就跑进去红色光圈里面，损失自身血量，开个回血的buff即可。重复此过程而已，非常容易简单。
  7. 修复能量塔！系统提示需要1个东西，人参果树（人参果）霸王龙（龙精）胆小鬼（小鬼胆)打对应的怪，霸王龙会眩晕大概0.1秒很烦人；胆小鬼会跑，使用控制技能即可。人参果树掉落3个东西，只有1个是“人参果”，其他2个是“入参果”，要注意看清楚！点击物品推到塔下即可。buff全加攻击和攻等，另外霸王龙建议远程职业不要靠近，后期的霸王龙可以秒人的，除非你够强大。（该关需注意不能一下把怪秒了，否则怪将不出东西）。
  8. 浮屠法阵！击杀须弥山神10个就能完成任务，如果杀死阿修罗众，任务进度就倒退一格。后期的此关卡对近战比较难，怪刷得快，阿修罗越积越多，根本抗不起。我后期遇到此关，直接重置，不喜欢打此关卡。

下面是9的倍数关：

  • 9、18关杀4大天王。建议先打魔礼青(半屏群伤)，顶着该boss在中间打，不要拉着走(为什么看后面)，boss放技能稍微跑动即可。其次打魔礼寿(毒)或者魔礼海（拉人），都啦到中间顶着打，最后魔礼红（火海）。为什么最好顶在在中间呢。拉人的时候把你啦到中间，魔礼青放群的时候，这个时候也比较好跑位。火海时，是躲不了的，要么开无敌，要么对着火海来的方向冲过去，一般不会造成伤害。我的无敌是用在最危险时使用，爆元或者无敌状态。
  • 27、36关BOSS，先杀小怪，得到百炼金拳，把箱子推到BOSS面前就行，后面BOSS把村名变成小怪打你，这时候BOSS无敌，先要杀完小怪，才能打BOSS；打完小怪，BOSS又会狂暴，反正躲就行了，等狂暴过去；没血了就去黄色光球哪里，会加血的！这个比较简单的。
  • 45、54关悲愤菩提，地上很多**(给boss回血)和地雷(禁锢boss)，就靠走位，引着boss踩地雷，禁锢了就揍他，他放群，就躲，慢慢磨死。
  • 63、72关boss拉人，飞人，群伤，出水晶是给玩家加血的。boss放群的时候，远程站在光圈外，揍他，近战只有等他放完群揍，然后学会跑动去吃水晶加血。
  • 81、90关消灭蛇魔女。打法很简单，BOSS会下蛇蛋，蛋蛋会孵出小蛇。下蛋了先快速打蛋蛋，孵出来的就是萎靡的小蛇，2下就死，不然就是活泼的小蛇，要难打一点点。后面还要出另外的2个小怪，也是先杀小的。有阵的职业比较好打，开阵群就是了。简单好打。
  • 99、108关，进去就去触碰绿球，然后打生命藤条，打到80%血量时，这个时候你就是5.0了，立马去打4个冒险者NPC，是平A，因为你是5.0，NPC就回血。最好保证4个NPC全救下，boss难度最小。打boss的时候，站在绿球那，出10秒怨灵超度，立马打生命藤条，80%以下血时，你就可以免疫boss的大招，只管揍boss即可。攻击低的话可以先打藤条一点点血，别低于80%，等到boss要开大招的时候， 再补1-2下就到80%以下，这样安全过关。
